# PracticHub

Учебный проект на C++/CLI и Windows Forms, в котором собраны 9 практик по объектно-ориентированному программированию и моделированию прикладных сценариев.

## Состав проекта

1. Практика 1. Класс инцидента безопасности: создание, валидация, чтение и сохранение в файл.
2. Практика 2. Анализатор инцидентов: загрузка списка инцидентов и ранжирование по уровню ущерба.
3. Практика 3. Поиск инцидентов: фильтрация по ключевым словам в описании.
4. Практика 4. Наследование: иерархия участников сервиса такси, машин и типов заказов.
5. Практика 5. Агрегация: заказ такси, маршруты, диспетчерский центр и состояние заказа.
6. Практика 6. Композиция и ассоциации: поездка, маршрут, платёжный шлюз и база оплат.
7. Практика 7. Контроллер: конечный автомат для оплаты по телефону или карте.
8. Практика 8. Приложение водителя: регистрация, выбор заказа, маршрут и завершение смены.
9. Практика 9. Логгирование: фабрика логгеров для консоли и файла.

## Структура репозитория

- `PracticHub/` — исходный проект Visual Studio.
- `practice_data/` — демонстрационные данные для практик 1, 2 и 3.
- `docs/listings/` — 9 отдельных листингов по каждой практике.
- `PracticHub.sln` — solution для Visual Studio.

## Сборка

1. Откройте `PracticHub.sln` в Visual Studio 2022.
2. Выберите конфигурацию `Debug | x64` или `Release | x64`.
3. Убедитесь, что установлен набор инструментов MSVC `v143` и .NET Framework `4.8.1`.
4. Запустите сборку решения.
